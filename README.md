# A 2D physics platformer prototype built in Unity where you control a blob-like character using slingshot movement, survive hazard traps, and progress through portals.

Slingshot movement:
Click and drag to aim
Release to launch the player with force

## Controls
Mouse Left Click + Drag: aim launch
Mouse Left Release: launch
Esc: return to Home scene from any non-Home scene
